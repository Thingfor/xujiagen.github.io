###opencv基本函数组成
opencv函数组成是：opencv、opencv2
opencv文件夹下面都是opencv1.0的函数组成
opencv2文件夹下面都是opencv2.0的函数组成

opencv2组成部分

组成 作用
calib3d 相机校准和三维重建相关，主要包括多视角几何算法，单个立体摄像头标定，物体姿态估计，立体相似性算法，3D信息的重组
contrib 新增的不太稳定的功能
core 核心功能模块：opencv基本数据结构、动态数据结构、绘图函数、数组操纵相关函数、辅助功能和系统函数和宏、与opengl相互操作
imgproc 线性与非线性图像滤波、图像的几何变换、其他图像变换、直方图相关、结构分析和形状描述、运动分析和对象跟踪、特征检测、目标检测
features2d 特征检测和描述、特征检测器通用接口、描述符匹配器通用接口、关键点描述函数和匹配功能绘制函数
flann 高纬度近似临近快速算法库：快速近似最近邻搜索、聚类
gpu 运用GPU加速计算机视觉计算
highgui 高层GUI图形用户界面
legacy 已经废弃的算法库：运动分析、期望最大化、直方图、平面细分、特征检测与描述、描述符提取器、通用描述符、匹配器
ml 统计模型、一般贝叶斯分类器、K-近邻、支持向量机、决策树、提升、梯度提高树、随机数、超随机数、期望最大化、神经网络、MLdata
nonfree 一些具有专利的算法模块
objdetect 目标检测模块
ocl 运用opencv加速计算机视觉组件模块
photo 包含图像修复和图像去燥两个部分
stiching 图像拼接模块

opencv还提供了官方的demo方便我们学习每一个模块，主要在opencv/sources/sample文件夹下面，有兼容各种编程语言的demo

mat简析：
保存图像和其他矩阵数据的数据结构

在opencv里面有几种输入的类型需要记忆：

名称 用法
string 输入一个字符串
INputArray 需要输入一个Mat数据类型的数据

需要记忆的变量：

名称|含义
WINDOW_AUTOSIZE|窗口的大小适应图片的大小
WINDOW_NORMAL|窗口的大小可以通过鼠标的标准
WINDOW_OPENGL|创建的窗口支持opengl

创建滑动条：createTrackbar()函数
第一个参数：轨迹条的名称
第二个参数：依附到创建的窗口上面的名称
第三个参数：指向整型的指针
第四个参数：滑动块可以达到的滑动块最大位置值
第五个参数：这是一个指向回调函数的指针，每一次滑块改变位置时，函数都会进行回调，函数原型必须void XXXX(int,void)第一个参数是轨迹条的位置，第二个参数是用户数据
第五个参数：这个参数是指向回调函数的数据
createtrackbar(TrackbarName(string),WINDOW_NAME,&滑块的初始位置(int类型),滑块的最大位置(int),函数名称(int,void))

setMouseCallback(窗口名称(string),on_MouseHandle(函数名称),&Mat指针)
