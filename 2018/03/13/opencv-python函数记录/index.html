###颜色空间
BGR/HSV/YUV/HSI/YCbCr
我们平时了解的是RGB色彩空间，RGB色彩空间也可以转换成HSV色彩空间，HSV色彩空间是圆锥形的，同时不同颜色有自己的区域，可以通过选择想要的区域区分我们需要的部分

颜色空间转换函数：cv.cvtcolor(变量,空间转换参数)

函数 空间转换函数 解析
cv.cvtColor(src,COLOR_BGR2GRAY) COLOR_BGR2GRAY 将RGB空间图像转换到灰度空间
cv.cvtColor(src,COLOR_RGB2HSV) COLOR_RGB2HSV 将RGB空间图像转换到HSV空间
cv.cvtColor(src,COLOR_RGB2YUV) COLOR_RGB2YUV 将RGB空间图像转换到YUV空间
cv.cvtColor(src,COLOR_RGB2YCrCb) COLOR_RGB2YCrCb 将RGB空间图像转换到YCrCb空间

opencv Rect矩阵类的用法
创建一个Rect对象的rect(100,50,50,100)，会有以下几个功能：
仔细看了一下Rect的源码，Rect是一个c++的矩阵类，创建一个矩阵类有很多方式，因为这个类里面有很多重载函数，方面我们用各种方式去创建一个矩阵类
Rect(x,y,height,width)，表示矩阵创建的x,y坐标，已经矩阵的宽度和高度
其中x,y表示的是左上点的坐标。
学会读源码，力量无穷大

名称 含义
rect.area() 返回area矩阵的面积
rect.size() 返回rect矩阵的尺寸
rect.tl() 返回左上定点的坐标
rect.br() 返回右下顶点的坐标
rect.width() 返回矩阵的宽度
rect.height() 返回矩阵的宽度
rect.contains(Point(x,y)) 返回布尔变量，是否包含某些点
rect=rect1 & rect2 求两个矩阵的交集
rect=rect1 \ rect2 求两个矩阵的并集
rect=rect+Point(-100,100) 将左上的点移动，x-100,y+100
rect=rect+size(-100,100) 左上的定点不变，宽度减少100，高度加上100
rect1==rect2 将矩阵进行对比，返回布尔变量

Mat E=Mat::eye(4,4,CV_64F) 按照matlab的形式创建数组

Mat RowClone=C.row(1).clone() 通过某一行创建另外一个Mat对象

CV_8UC3创建的图像是uint8，channel3形式的图片，其他的可以以此类推

Mat M(7,7,CV_32FC2,Scalar(1,3)); 创建了一个7*7的数组，float32数据组成，双通道，Scalar说明通道是(1,3)组成的，第一个通道里面全部是1，第二个通道里面全部是3

如果创建的是多个通道的，同时每一个通道里面是数据都是一样的，可以用Scalar:all(num)向所有的通道里面填充数据
