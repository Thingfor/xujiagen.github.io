shadowsocks创建ssr节点

发表于 2018-03-05 | 分类于 shadowsocks

购买服务器

首先要说的是购买服务器，购买服务器我建议是购买vultr上面的服务器，是按照小时计费的，如果节点被封了之后，可以直接更换节点，当然敏感时期直接封掉IP段，那是另外一种情况。

安装shadowsocks

在配置完服务器之后（不说明这一点，太基本了），不同服务器可能需要安装wget
我是在centos 7上面安装的，所以直接按照centos 7指令进行说明。

一般centos会安装wget，其他服务器如果没有安装wget可能需要首先安装wget
相关指令是：
yum install wget

之后是下载.sh文件：
wget –no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocksR.sh

赋予相关文件权限：
chmod +x shadowsocksR.sh

运行.sh文件：
./shadowsocksR.sh 2>&1 | tee shadowsocksR.log

一般建立的shadowsocks节点的速度会比较慢，会安装加速模块
我这里主要安装谷歌BBR加速器：
wget –no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh
这里是下载安装BBR加速器的.sh文件

赋予文件运行权限：
chmod +x bbr.sh

安装BBR：
./bbr.sh
