imgproc模块介绍

发表于 2018-03-21 | 分类于 opencv

imgproc模块是一个很大模块

里面包含的内容有：
三种线性滤波：方框滤波、均值滤波、高斯滤波
两种非线性滤波：中值滤波、双边滤波
七种图像处理形态学：腐蚀、膨胀、开运算、闭运算、形态学梯度、顶帽、黑帽
漫水填充
图像缩放
图像金字塔
阈值化

首先需要说明得是平滑处理：

平滑处理就是模糊处理，是一种在图像处理前期非常常见的一种现象。主要是减少图像上面的噪点和失真。
处理的方法是使用图像滤波器：

在图像中，图像的能量主要存储在幅值谱的低频和中频段，在较高的频段，有用的信息经常被淹没，因此一个降低高频成分的滤波器能够减弱噪声的影响。

图像处理的目的有两个：

抽出对象的特征作为图像识别的特征模式；
另一个是适应出现处理的要求，减低图像中混入的噪声。

图像处理的要求也有两个:

一是不影响图像的轮廓和边缘的特性
二是让图像清晰视觉效果好。

常用的opencv中有五个常用的图像平滑处理的操作：

名称 形式
方框滤波 BoxBlur函数
均值滤波 Blur函数
高斯滤波 GaussianBlur函数
中值滤波 medianBlur函数
双边滤波 bilateraFilter函数

线性滤波器:

名称 解释
低通滤波器 允许低频率通过
高通滤波器 允许高频率通过
带通滤波器 允许一定范围里面的频率通过
带阻滤波器 阻止一定频率通过允许其他频率通过
全通滤波器 允许所有频率通过，仅仅改变相位关系
陷波滤波器 阻止一个狭窄频率范围通过，是一种特殊带阻滤波器

解释一下滤波和模糊之间的关系

滤波可以分为高通滤波和低通滤波两种，低通就是模糊，高通就是锐化

方框滤波函数

BoxBlur(输入图像，输出图像，深度，卷积核的尺寸，锚点(被平滑的那个点)，内核是否被其局域归一化，用于推断图像像素的边界模式)
其中图像的深度ddepth=-1表示图像的深度和原来的深度一样
卷积核的尺寸：size(3,3)卷积核的大小是一个3*3的矩阵
锚点表示被平滑的那个点，point(-1,-1)表示卷积核中心的那个点

高斯滤波是一种线性平滑滤波，可以消除高斯噪声，广泛应用与图像处理的减噪过程

其他函数调用的方式已经调用的形式基本上是相似的。

线性滤波:方框滤波、均值滤波、高斯滤波都是线性滤波
非线性滤波：中值滤波、双边滤波
线性滤波器主要是指两个信号之间的响应和他们各自的响应之和相等
非线性滤波对去除脉冲噪声、椒盐噪声有非常好的效果

中值滤波器：如果是3*3的窗函数，以点(i,j)为函数的中心，这个9个点的中值作为这个点的值所以在保存边缘还是消除噪声方面都有非常好的效果

腐蚀和膨胀

腐蚀和膨胀的功能在于：

1.消除噪声
2.分割出独立的图像元素，在图像中链接相邻元素
3.寻找图像中明显的极大值区域和极小值区域
4.求出图像梯度

前言：膨胀和腐蚀都是针对图像中的白色区域而不是黑色区域
膨胀：相当于图像中白色的区域被膨胀，白色所占有的面积更大了
通过一个确定大小的算子计算一个区域的最大值，将这个最大值保存给这个区域的中心，来实现白色区域的膨胀
腐蚀：图像中白色的区域被腐蚀，面积更小了
通过计算几个区域的最小值，将这个区域的最小值保存在这个区域的中心，实现白色区域的腐蚀。

开运算

就是先腐蚀在膨胀：开运算可以消除小物体，在纤细的点处分离物体，并且平滑在较大物体的边界的同时不明显改变他的面积。

闭运算

先膨胀在腐蚀，闭运算可以排除小型黑洞

形态学梯度

就是膨胀图和腐蚀图只差

顶帽

开运算的结果是放大了裂缝或者局部低亮度的区域

黑帽

是闭运算的结果图与与原图的差

在opencv里面因为这几个操作都是输入原图像，输出处理之后的图像，设置矩阵内核的大小，所以通过一个morphologyEx来实现

漫水填充

漫水填充是通过一种特定的颜色填充联通的区域，通过设置联通区域的上下限以及联通方式来达到不同的填充效果。

图像金字塔

以前经常在论文里面看到，今天终于有幸开始制作高斯金字塔了。

金字塔的底层是最原始的图像，往上则是低分辨率的金字塔

论文里面出现的金字塔主要分成两种：

高斯金字塔
拉普拉斯金字塔

高斯金字塔

1.对图像Gi进行高斯内核卷积
2.将所有偶数行和列去除

高斯图像里面有一个resize函数主要用来修改图像的尺寸
resize(src,dst,dsize,fx,fy,插值方式)
第一个参数是输入的原始图像
第二个参数是输出的图像
第三个参数是输出的尺寸，这个参数和下面的两个参数是矛盾的，要么填写这个参数，要么填写下面两个蚕食
第四个参数：图像在水平方向的缩放系数
第五个参数：图像在垂直方向的缩放系数
第六个参数：图像的差值方式
通过这个程序可以将图像的尺寸增大，减小

插值方式 含义
INTER_NEAREST 最近插值法
INTER_LINEAR 线性插值法
INTER_AREA 区域插值
INTER_CUBIC 区域插值
INTER_LANCZOS4 插值

阈值化
阈值化在作用在于在一幅图片上面利用阈值分割出我们想要的物体部分
固定阈值操作：典型的是对灰度图像进行阈值操作得到二值化图像，或者去掉噪声。
double threshold(输入图像，输出图像，阈值的具体值，不懂)

自适应与阈值化操作：adaptiveThreshold函数
void adaptiveThreshold(输入图像，输出图像，给像素赋满足条件的非零值，用于设置指定的自适应算法)
