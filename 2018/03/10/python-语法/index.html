函数的参数

1.位置参数 (传入元素的数量与形参的数量是相同的)
2.默认参数 (将一些参数设定默认值，如果这些参数没有传入，则使用默认值)
注意： 默认参数需要注意必须设定成一个不变的对象
注意： 不变对象,int型数据，str型数据,None型数据

3.可变参数 (传进去的参数是可变的)

这个函数在调用的时候需要组装一个list或者tuple

calc([1,2,3,4]) #调用的是位置参数

如果需要传入可变参数可以写成

对于想要输入一个list或者是tuple，可以用calc(*number)加上一个星号的形式，传入的参数可以是0个，也可以很多

4.关键字参数：

def person(name,age,**kw):
print(‘name:’,name,’age:’,age,’other:’,kw)

关键字参数允许你传入0个或者任意个数的参数，这些参数会组装成一个dict(字典)
传入的形式是：

person(‘bob’,35,city=’beijing’)

也可以创建一个dict,然后把该dict转换成关键字传进去

extra={‘city’:’beijing’,’job’:’engine’}
person(“bob”,35,**extra)

直接以字典的形式传进去

5.命名关键字参数：

限制传入的关键字的名字（命名关键字参数只需要一个*,就可以将其后面的元素设定成命名关键字参数）
命名关键字参数必须传入参数名，因为没有传入关键字名称，调用就会出错，但是可以有适当去缺省值

6.组合参数

在定义过程中可以将必选关键字参数，默认关键字参数，可变参数，关键字参数和命名关键字参数相组合，关键字的顺序是：必选参数，默认参数，可变参数，命名关键字参数，和关键字参数

生成器

生成器与列表生成器的区别在于，生成器生成的数据可以非常多，但是不会占用大量的内存，因为他是实时计算出来的，使用的时候才开始调用。

生成器的使用方式不仅仅是像列表生成器那样通过一个[]，对于一个函数如果将函数里面的数据通过yield操作，这个函数就相当于一个生成器

def fib(max):
n,a,b=0,0,1
while n<max:
yield b
a,b=b,a+b
n+=1

装饰器：

def log(func):

return warpper

@log

注意：这里首先定义了一个装饰器，这个装饰器需要输入一个函数，但是我们在创建新的函数的时候，在函数定义的开头创建一个@装饰器名称，相当于：装饰器函数(刚创建的函数)，就会在刚创建的函数运行之后返回一个调用装饰器函数的结果
对于装饰器的理解，我百度了下

理解：程序本身就像内裤，内裤非常重要，执行一些非常重要的操作，但是不能一直穿，至少冬天穿的时候非常冷，于是就有了裤子，裤子把内裤包裹在里面的同时，提供的避寒的功能，这个裤子就是装饰器，在不影响函数整体运行的同时创建装饰器，很方便。

注意：创建一个装饰器名称是log,创建一个函数是now()
执行now()函数相当于执行 log(now)()函数

###面向对象编程

在python的class中如果想要内部属性不被外部的所访问，可以在属性前面加上两个下划线，以两个下划线开头的变量名就是一个私有变量

私有变量只能class内部进行访问，在外部是不能访问的

所以可以将刚才的程序进行修改：

class Student(object):
def.init(self,name,score)
self.name=name
self.score=score

def print_score(self):
    print('%s:%s' %(self.__name,self.__score))

这样class内部本来的两个共有变量就变成的私有变量

需要注意的是以双下划线开头并且以双下划线结束的变量是class内部的特殊变量，特殊变量是可以直接访问的。

当看到单下划线开头的变量虽然可以直接访问，但是不建议随便访问，因为别人将名称修改成这样就是叫你不要求访问他

###继承和多态

在oop程序设计中，当我们定义一个class的时候，可以从某个现有的class继承，新的class称之为子类，这个和c++上面也是一样的，被继承的称之为基类或者说是父类

首先我们创建一个class:

class Animal(object):
def run(self):
print(“animal is running”)

class Dog(Animal):
pass

class Cat(Animal):
pass

我们首先创建了一个对象class，然后创建两个class继承第一个class,他的好处在于可以直接继承第一个class上面所有的内容
dog Dog()
dog.run()
创建一个对象dog,因为Dog已经从Animal上面继承了所以直接运行dog.run()就可以运行Animal的子程序

当子类和父类都存在相同的代码run()的时候，我们说子类run()覆盖了父类的run()，在代码运行的时候，总是会调用子类的run(),这样我们就获得的继承的另外一个好处，多态

isinstance()函数可以判断一个数据类型是不是一样的，这样当我们创建一个cat的类，同时这个类是继承父类animal，这个时候，判断一个一个cat类型的对象是否与animal的数据类型相同的时候就会发现，cat的队形与animal的数据类型是一样的

但是反过来就不可以

我们创建一个函数调用animal对象

def run_twice(Animal):
Animal.run()
Animal.sun()

这个时候我们传入run_twice(Animal()) //注意传入的都是实例
都是调用各自的run函数，虽然都是从Animal继承run()函数，但是当继承的子类内部创建了run()函数，就会屏蔽父类的run()函数，体现class的多态

这个时候其实是两种编程语言之间的区别：
静态语言和动态语言
静态语言：需要传入animal类型和animal类型的子类否则无法调用run函数
动态语言：不一定需要传入animal类型的语言，只要保证传入的对象有一个run()函数
