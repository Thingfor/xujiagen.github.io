python 函数复习，最近需要写一写python程序做毕设

函数的参数

1.位置参数 (传入一个简单程序的参数，需要多少个，，传入多少个)
2.默认参数 (将一些参数设定默认值，如果这些参数没有传入，则使用默认值)
注意： 默认参数需要注意必须设定成一个不变的对象
注意： 不变对象,int型数据，str型数据,None型数据

3.可变参数 (传进去的参数是可变的)

def calc(number):
sum=0
for n in number:
sum+=n
return sum

这个函数在调用的时候需要组装一个list或者tuple

calc([1,2,3,4]) #调用的是位置参数

如果需要传入可变参数可以写成

def calc(*number):
sum=0;
for n in number:
sum+=n
return sum

这个可变参数的函数输入是：

calc(1,2,3,4)

对于想要输入一个list或者是tuple，可以用calc(*number)加上一个星号的形式，传入的参数可以是0个，也可以很多

4.关键字参数：

def person(name,age,**kw):
print(‘name:’,name,’age:’,age,’other:’,kw)

关键字参数允许你传入0个或者任意个数的参数，这些参数会组装成一个dict(字典)
传入的形式是：

person(‘bob’,35,city=’beijing’)

也可以创建一个dict,然后把该dict转换成关键字传进去

extra={‘city’:’beijing’,’job’:’engine’}
person(“bob”,35,**extra)

直接以字典的形式传进去

5.命名关键字参数：

限制传入的关键字的名字（命名关键字参数只需要一个*,就可以将其后面的元素设定成命名关键字参数）
命名关键字参数必须传入参数名，因为没有传入关键字名称，调用就会出错，但是可以有适当去缺省值

6.组合参数

在定义过程中可以将必选关键字参数，默认关键字参数，可变参数，关键字参数和命名关键字参数相组合，关键字的顺序是：必选参数，默认参数，可变参数，命名关键字参数，和关键字参数
