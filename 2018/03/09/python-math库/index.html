发表于 2018-03-09

对于python math库是平时比较常用的库，现在总结一下，以后方面查找

子函数列表 描述 例子
ceil(x) 去大于等于x的最小整数，如果是整数，返回整数 ceil(4.01)=5
copysign(x,y) 将第二个的符号加到第一个上面去 copysign(4,-5)=-4
cos(x) 求余弦，输入值必须是弧度 cos(math.pi/2)=0
degrees(x) 将弧度转换成角度 math.degrees(math.pi)=180.0
e 常量e 2.718281828459045
exp(x) e^x e的多少次方
expm1(x) exp_minus_1,求到的值结果减1 math.expm1(1)=1.71
fabs(x) 返回x的绝对值 math.fabs(-0.003)
factorial(x) x!阶乘 3！=6
floor(x) 返回小于等于x的最大整数 4.001=4
fmod(x,y) 得到x/y的余数 fmod(9,2)=1
frexp(x) 什么鬼东西 不看，哼
fsum(iterator) 对迭代器里面的元素求和 fsum([1,2,3,4]
gcd(x,y) 返回x和y的最大公约数 gcd(8,6)=2
hypot(x,y) x^2+y^2 返回x和y的平方和
isfinite(x) 检测x是不是无限大的数，返回布尔类型
isinf(x) 判断是不是无穷大
isnan(x) 判断x是不是数字not a number isnan(23)=false
ldexp(x,i) 返回x*(2^i) math.ldexp(5,5)=160
log(x) 默认底数为e
log10(x) 返回x是以10为底数的
log1p(x) 返回x+1的自然对数值log(x+1)
log2(x) 返回基是以2为对数的 math.log2(32)=5.0
modf(x) 返回小数部分和整数部分的组成元组 modf(12,.001)=(0.001,12)
pow(x,y) 返回x的y次方 pow(3,4)=81.0
radians(x) 把角度转变成弧度 radians(45)=0.7853981
sin(x) 求x的正弦值
sqrt(x) 求x的平方根 sqrt(4)=2
