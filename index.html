首先需要说明得是平滑处理：

平滑处理就是模糊处理，是一种在图像处理前期非常常见的一种现象。主要是减少图像上面的噪点和失真。
处理的方法是使用图像滤波器：

在图像中，图像的能量主要存储在幅值谱的低频和中频段，在较高的频段，有用的信息经常被淹没，因此一个降低高频成分的滤波器能够减弱噪声的影响。

图像处理的目的有两个：

抽出对象的特征作为图像识别的特征模式；
另一个是适应出现处理的要求，减低图像中混入的噪声。

图像处理的要求也有两个:

一是不影响图像的轮廓和边缘的特性
二是让图像清晰视觉效果好。

常用的opencv中有五个常用的图像平滑处理的操作：

线性滤波器:

名称 解释
低通滤波器 允许低频率通过
高通滤波器 允许高频率通过
带通滤波器 允许一定范围里面的频率通过
带阻滤波器 阻止一定频率通过允许其他频率通过
全通滤波器 允许所有频率通过，仅仅改变相位关系
陷波滤波器 阻止一个狭窄频率范围通过，是一种特殊带阻滤波器

解释一下滤波和模糊之间的关系

滤波可以分为高通滤波和低通滤波两种，低通就是模糊，高通就是锐化

腐蚀和膨胀

腐蚀和膨胀的功能在于：

1.消除噪声
2.分割出独立的图像元素，在图像中链接相邻元素
3.寻找图像中明显的极大值区域和极小值区域
4.求出图像梯度

前言：膨胀和腐蚀都是针对图像中的白色区域而不是黑色区域
膨胀：相当于图像中白色的区域被膨胀，白色所占有的面积更大了
通过一个确定大小的算子计算一个区域的最大值，将这个最大值保存给这个区域的中心，来实现白色区域的膨胀
腐蚀：图像中白色的区域被腐蚀，面积更小了
通过计算几个区域的最小值，将这个区域的最小值保存在这个区域的中心，实现白色区域的腐蚀。

开运算

就是先腐蚀在膨胀：开运算可以消除小物体，在纤细的点处分离物体，并且平滑在较大物体的边界的同时不明显改变他的面积。

闭运算

先膨胀在腐蚀，闭运算可以排除小型黑洞

形态学梯度

就是膨胀图和腐蚀图只差

顶帽

开运算的结果是放大了裂缝或者局部低亮度的区域

黑帽

是闭运算的结果图与与原图的差

在opencv里面因为这几个操作都是输入原图像，输出处理之后的图像，设置矩阵内核的大小，所以通过一个morphologyEx来实现

isinstance()函数可以判断一个数据类型是不是一样的，这样当我们创建一个cat的类，同时这个类是继承父类animal，这个时候，判断一个一个cat类型的对象是否与animal的数据类型相同的时候就会发现，cat的队形与animal的数据类型是一样的

函数的参数

1.位置参数 (传入一个简单程序的参数，需要多少个，，传入多少个)
2.默认参数 (将一些参数设定默认值，如果这些参数没有传入，则使用默认值)
注意： 默认参数需要注意必须设定成一个不变的对象
注意： 不变对象,int型数据，str型数据,None型数据

3.可变参数 (传进去的参数是可变的)

def calc(number):
sum=0
for n in number:
sum+=n
return sum

这个函数在调用的时候需要组装一个list或者tuple

calc([1,2,3,4]) #调用的是位置参数

如果需要传入可变参数可以写成

def calc(*number):
sum=0;
for n in number:
sum+=n
return sum

这个可变参数的函数输入是：

calc(1,2,3,4)

对于想要输入一个list或者是tuple，可以用calc(*number)加上一个星号的形式，传入的参数可以是0个，也可以很多

4.关键字参数：

def person(name,age,**kw):
print(‘name:’,name,’age:’,age,’other:’,kw)

关键字参数允许你传入0个或者任意个数的参数，这些参数会组装成一个dict(字典)
传入的形式是：

person(‘bob’,35,city=’beijing’)

也可以创建一个dict,然后把该dict转换成关键字传进去

extra={‘city’:’beijing’,’job’:’engine’}
person(“bob”,35,**extra)

直接以字典的形式传进去

5.命名关键字参数：

限制传入的关键字的名字（命名关键字参数只需要一个*,就可以将其后面的元素设定成命名关键字参数）
命名关键字参数必须传入参数名，因为没有传入关键字名称，调用就会出错，但是可以有适当去缺省值

6.组合参数

在定义过程中可以将必选关键字参数，默认关键字参数，可变参数，关键字参数和命名关键字参数相组合，关键字的顺序是：必选参数，默认参数，可变参数，命名关键字参数，和关键字参数
